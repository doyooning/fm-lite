# 00. 프로젝트 개요

## 게임 콘셉트

- 가상 팀 8개 / 팀당 18명 가상 선수
- 8강 싱글 엘리미네이션 토너먼트 (8강 → 4강 → 결승, 사용자 기준 최대 3경기)
- 사용자는 팀 선택 → 상대 분석 → 전술 설정 → 텍스트 경기 시뮬레이션 → 경기 중 선택지 대응 → 결과 확인을 반복하며 우승에 도전

## 핵심 게임 루프

```
새 게임 시작 (팀 선택)
   │
   ▼
┌─────────────────────────────────────┐
│ 1. 다음 경기 확인                     │
│ 2. 상대 팀 분석 조회                  │
│ 3. 경기 전 전술 설정                  │
│ 4. 경기 시뮬레이션 시작                │
│    ├─ 이벤트 텍스트 스트림             │
│    └─ 선택지 이벤트 발생 → 사용자 선택  │
│ 5. 경기 종료 → 결과 저장              │
│ 6. 같은 라운드 AI vs AI 경기 자동 처리 │
└───────────────┬─────────────────────┘
                │ 승리          │ 패배
                ▼               ▼
        다음 라운드 진행      게임 오버 (탈락)
                │
                ▼ (결승 승리)
              우승 🏆
```

## MVP 범위 (필수 기능)

| # | 기능 | 비고 |
|---|---|---|
| 1 | 팀 목록/상세 조회 | 마스터 데이터 |
| 2 | 선수단 조회 | 능력치 8종 + 특성 |
| 3 | 새 게임 시작 / 선택 팀 저장 | SaveGame 생성 |
| 4 | 대회 생성 | 새 게임 시작 시 자동으로 8강 대진 생성 |
| 5 | 다음 경기 조회 | 사용자 팀의 다음 경기 |
| 6 | 상대 팀 분석 조회 | 규칙 기반 요약 (강점/약점/예상 전술) |
| 7 | 경기 전 전술 설정 | formation / mentality / pressing / lineHeight / attackStyle |
| 8 | 경기 시뮬레이션 | 규칙 기반 엔진, 텍스트 이벤트 |
| 9 | 경기 중 선택지 처리 | 경기당 2~3회, 선택이 이후 흐름에 영향 |
| 10 | 경기 결과 저장 | 스코어 + 통계 + 이벤트 로그 |
| 11 | 토너먼트 진행 관리 | 라운드 종료 시 다음 라운드 대진 생성 |
| 12 | 우승 처리 | 결승 승리 시 CHAMPION 상태 |

## MVP에서 의도적으로 제외하는 것

- 회원가입/로그인 (익명 user 자동 생성, 확장 시 JWT)
- 선수 교체/부상/징계/성장/이적
- 경기 간 체력·컨디션 이월 (경기마다 컨디션 랜덤 생성으로 단순화)
- 선발 라인업 수동 편집 (포메이션에 따라 능력치 기준 베스트 XI 자동 선발)
- LLM 기반 리포트/해설 (확장 기능)
- 무승부 리그전 (토너먼트만, 동점 시 간이 승부차기)

## 역할 구분

| 역할 | 담당 |
|---|---|
| 경기 판정 (골, 찬스, 결과) | Spring Boot Rule Engine (결정적 규칙 + 시드 기반 랜덤) |
| 상대 팀 전술 변화 | 상태 기반 규칙 AI (조건 → 전술 조정 테이블) |
| 경기 설명 / 분석 문장 | MVP: 템플릿 문자열 → 확장: LLM 생성 |
